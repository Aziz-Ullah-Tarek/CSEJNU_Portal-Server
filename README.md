# CSE JNU Backend

Backend server for CSE @ JNU Portal

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file with required environment variables

3. Run development server:
```bash
npm run dev
```

4. Run production server:
```bash
npm start
```

## API Endpoints

- `GET /` - Welcome message
- `GET /api/health` - Health check endpoint

## Technologies

- Node.js
- Express.js
- CORS
- dotenv
